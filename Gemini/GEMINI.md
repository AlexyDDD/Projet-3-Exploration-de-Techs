# Directory Overview

This directory serves as a comprehensive knowledge base for learning and using the n8n workflow automation platform. It is structured around a central learning plan and contains detailed documentation, practical project ideas, step-by-step workflow tutorials, and a curated list of external learning resources.

# Key Files

*   **`n8n_learning_plan.md`**: The core document that outlines a structured, step-by-step guide to learning n8n, from beginner to advanced topics.
*   **`n8n_etape1_documentation.md`**: A deep-dive guide into the fundamental concepts of n8n, covering its data structure, execution flow, and core components.
*   **`n8n_etape6_documentation.md`**: A detailed document explaining advanced n8n topics, including error handling strategies, using the Code Node, implementing webhooks, and an introduction to custom node development.
*   **`workflow_creation_explanation.md`**: A file containing several ideas for simple, practical workflows that can be built with n8n to solve common automation problems.
*   **`workflow_1_auto_social_post.md`**: A step-by-step tutorial for building the "Automated Social Media Posting" workflow, from reading an RSS feed to posting on Twitter.
*   **`workflow_2_lead_capture.md`**: A step-by-step tutorial for building the "Lead Capture and Notification" workflow, from receiving webhook data to saving it in Google Sheets and notifying a team on Slack.
*   **`workflow_3_scheduled_report.md`**: A step-by-step tutorial for building the "Scheduled Data Report" workflow, from running on a schedule to fetching data, processing it, and emailing a summary.
*   **`n8n_learning_resources.md`**: A curated list of the best free, digital resources (official docs, YouTube channels, community forums, blogs) for learning n8n.

# Usage

This directory is intended to be used as a self-contained educational resource for mastering n8n.

1.  **Start with `n8n_learning_plan.md`** to get an overview of the learning path.
2.  Refer to the detailed documentation files (`n8n_etape1_documentation.md`, `n8n_etape6_documentation.md`) as you progress through the corresponding steps in the plan.
3.  Use the ideas in `workflow_creation_explanation.md` for inspiration, and follow the detailed tutorials (`workflow_1`, `workflow_2`, etc.) as practical exercises to solidify your understanding.
4.  Consult `n8n_learning_resources.md` to find external tutorials, get help from the community, or explore topics in greater depth.
