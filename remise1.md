## Sujet choisi
Nous avons choisi d’explorer **n8n**, un outil d’automatisation open-source permettant de connecter différentes applications, services web et API entre eux, sans nécessiter de code complexe.

L’objectif de notre projet est de :
- Comprendre les principes d’automatisation avec **n8n**.
- Expérimenter la création de **workflows automatisés** (ex. : intégrations entre messagerie, base de données, services cloud, etc.).
- Utiliser l’intelligence artificielle (ChatGPT, GitHub Copilot, etc.) pour nous aider à :
  - Comprendre les fonctionnalités de l’outil,
  - Corriger des erreurs de configuration,
  - Générer de la documentation claire et cohérente.
